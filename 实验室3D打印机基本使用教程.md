

# 1.下载

关于设备的连接，当你在连接实验室WLAN时，3D打印机自动连接。在**Bambu Studio**中，上面一行点击设备，即可看到如下所示（点击加号就能看见）

# 4.导入文件

 现在你已经做好打印前准备，将sw文件直接拖入软件里（下图里的右边区域）

注：在拖入之前见零件的类型改为STEP开头。

```
   如果打印的模型有孔洞，因为耗材融化，向内坍塌，会比预计的小，所以要将其在导入之前孔洞直径增加0.20mm。
```

# 5.调设基本设置及处理

## （1）工艺

点击准备，在工艺打开高级并在下面的选择栏改为如图（0.20mm Strength @BBL A1）

![屏幕截图 2026-04-30 143239](https://raw.githubusercontent.com/456asdb456/my-image-bed/main/images/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-04-30%20143239.png)

## （2）强度

如果对强度有要求，则在点击强度后，往下拉调整稀疏填充密度

## （3）支撑

如果打印物品，有浮空地方，则要点击支撑，将页面调整为如下

![屏幕截图 2026-04-30 150417](https://raw.githubusercontent.com/456asdb456/my-image-bed/main/images/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-04-30%20150417.png)

## （4）打印

点击右上角打印切盘，弹处的提示默认即可，等待打印结束，完成后用小铲子，铲下来。（注：板子很烫的）

在下次打印前，把板子拿下来散会热。

## （5）清洁

打印后，要把板上残留除干净。
