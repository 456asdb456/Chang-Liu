# 4.

 现在你已经做好打印前准备，将sw文件直接拖入软件里（下图里的右边区域）

注：在拖入之前见零件的类型改为STEP开头。

       如果打印的模型有孔洞，因为耗材融化，向内坍塌，会比预计的小，要将其孔洞增加0.20mm。![屏幕截图](https://raw.githubusercontent.com/456asdb456/my-image-bed/main/images/%E5%B1%8F%E5%B9%95%E6%88%AA%E5%9B%BE%202026-04-29%20225859.png)
